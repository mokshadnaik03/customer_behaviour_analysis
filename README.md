# customer_behaviour_analysis
Data Analytics Project showing customer behavior analysis using Python, SQL, PowerBI

Overview

This project analyzes customer purchase data to understand shopping patterns, customer segments, and product performance. The goal is to generate insights that help businesses make better marketing and sales decisions.

Dataset
~3,900 transaction records
18 columns including customer demographics, purchase details, discounts, ratings, and shipping information.

Tools & Technologies
Python (Pandas, NumPy)
SQL (PostgreSQL/MySQL)
Power BI
Gamma (Presentation)

Steps Performed
Loaded and explored the dataset using Python.
Cleaned data and handled missing values.
Performed feature engineering (age groups, purchase behavior).
Stored processed data in SQL.
Wrote SQL queries to analyze revenue, customers, and products.
Built an interactive Power BI dashboard.

Key Insights
Identified top-selling and top-rated products.
Compared subscriber vs non-subscriber spending.
Analyzed revenue by gender and age group.
Found high-spending discount users.

Dashboard
Power BI dashboard visualizes:
Customer count
Revenue by category
Sales by age group
Subscription distribution

Outcome
The project demonstrates skills in data cleaning, SQL analysis, and business intelligence visualization.
